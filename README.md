# edgy_board number b142
This design is part of the [edgy board project](https://github.com/skunkforce/edgy_boards). It provides a waterproof USB 2.0 capable 16 pin USB C connector.

[LCSC Part C3020043](https://lcsc.com/product-detail/USB-Connectors_SHOU-HAN-TYPE-C-16PFS-4J-H7-05-IPX6_C3020043.html)

# getting started
This repository uses submodules. After cloning use the command 

```$ git submodule update --init --recursive```

to pull the submodules before opening the project with kicad.
